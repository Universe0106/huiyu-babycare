# 慧育宝宝 1.0 版本需求大纲

## 一、产品概述

**产品名称**：慧育宝宝
**产品类型**：微信小程序
**版本**：1.0
**定位**：新手爸妈的AI育儿助手，提供专业、便捷的育儿知识问答与成长记录服务

---

## 二、目标用户

| 特征 | 描述 |
|------|------|
| 核心用户 | 0-3岁宝宝的父母（尤其是新手爸妈） |
| 痛点 | 育儿经验不足，面对宝宝问题焦虑，需要快速获取专业建议 |
| 需求 | 快速问答、成长记录、每日知识推送 |

---

## 三、功能模块

### 3.1 核心功能

#### 3.1.1 AI 问答对话
- **功能描述**：用户通过对话方式向AI提问育儿问题，AI结合知识库给出专业解答
- **问答类型**：
  - 症状处理类：发烧、红屁股、湿疹、腹泻、便秘、感冒等
  - 喂养指导类：母乳/奶粉喂养、辅食添加、奶量评估
  - 日常护理类：脐带护理、口腔护理、睡眠习惯、耳朵护理
  - 发育监测类：身高体重对照、大运动发展里程碑
  - 健康防护类：疫苗接种、常见用药指导
- **答案形式**：文字解答 + 建议 + 注意事项（必要时提示就医）
- **多模态支持**：未来支持图片问诊（1.0仅文字）

#### 3.1.2 宝宝成长记录
- **宝宝信息管理**：
  - 添加宝宝：姓名、性别、出生日期、头像
  - 支持多宝宝（最多3个）
  - 宝宝切换功能
- **生长数据记录**：
  - 身高、体重、头围
  - 记录时间
  - 备注说明
- **成长曲线展示**：
  - 折线图展示身高/体重变化
  - 对照WHO国际标准（0-3岁生长曲线）
  - 当前发育水平评估（正常/偏轻/偏重/偏高/偏矮）

#### 3.1.3 知识库浏览
- **分类导航**：
  - 日常护理（红屁股、脐带护理、湿疹、睡眠、耳朵护理、口腔护理）
  - 常见症状（发烧、感冒、皮疹、腹泻、便秘）
  - 发育监测（身高体重、大运动）
  - 喂养指导（奶粉选择、辅食添加）
  - 健康防护（疫苗接种、常见用药）
- **文章列表**：点击分类查看该分类下的所有文章
- **搜索功能**：关键词搜索知识库文章

#### 3.1.4 每日推送
- **推送内容**：每日一篇精选育儿知识文章
- **推送时间**：用户可设置（默认早9点）
- **推送形式**：小程序内 banner + 通知（需用户授权）
- **内容来源**：知识库精选

#### 3.1.5 用户中心
- **宝宝管理**：添加/编辑/删除宝宝
- **设置**：通知开关、推送时间设置
- **使用记录**：问答历史、成长记录历史

## 六、技术架构（1.0）

### 6.1 前端
- **框架**：Taro 3.x + React
- **请求**：原生 Taro.request
- **状态管理**：React Hooks（useState/useContext）
- **图表**：ECharts（成长曲线）

### 6.2 后端
- **框架**：Express.js
- **存储**：内存存储（1.0）/ MySQL（后续版本）
- **AI对接**：知识库 + 关键词匹配（1.0）/ 向量检索（后续）

### 6.3 知识库
- **来源**：飞书知识库（内容需要录入）
- **形式**：QA对 + 文章
